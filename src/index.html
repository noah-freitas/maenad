Maenad

Welcome to Maenad, a music player.

Maenad plays songs in your browser. You can currently get songs into Maenad from two sources:

  • Local Files: Use the file input to import one or more MP3s from your computer.
  • AWS (S3): Use the configuration page to set connection variables, then go to the load page. You should see the list of the files in your bucket. Click on each one you want to import.
